Welcome to AI Docs Vector DB Hybrid Scraper documentation!
===========================================================

A comprehensive documentation scraping system combining Crawl4AI (bulk) + Firecrawl MCP (on-demand) 
with Qdrant vector database for Claude Desktop/Code integration.

Features
--------

* **Hybrid scraping**: Bulk processing with Crawl4AI and on-demand with Firecrawl
* **Vector search**: Powered by Qdrant vector database
* **Claude integration**: MCP server for Claude Desktop/Code
* **FastAPI backend**: Modern async Python web framework
* **Comprehensive monitoring**: Prometheus metrics and Grafana dashboards
* **Flexible deployment**: Docker, Docker Compose, and Kubernetes support
